# USB_Stealer
A U-Disk spy progarm for myself learning to C# and .NET framework.

***

## 如何使用
打开即用

## 配置
在选项框中写上 目标路径 和 扩展名列表 。

目标路径默认为：D:\1UsbStealerPlatinum
扩展名列表默认为：.doc;.docx;.mp3

扩展名列表格式应符合 **扩展名;扩展名** 的形式，使用 **;** 分隔扩展名
